## API Reference

#### Get documentation

```http
    https://dictionaryapi.dev/
```

#### Get item

```http
    GET https://api.dictionaryapi.dev/api/v2/entries/en/${word}
```

| Parameter | Type     | Description                                       |
| :-------- | :------- | :------------------------------------------------ |
| `word`    | `string` | **Required**. the word of which definition needed |
